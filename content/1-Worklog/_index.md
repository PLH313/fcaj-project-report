**Week 1:** [Foundation Setup: Cost Management, IAM Security, VPC Networking, EC2, and Cloud9](1.1-week1/)

**Week 2:** [Deployment: S3, RDS, Lightsail, Containerization, Auto Scaling, CloudWatch, and DynamoDB](1.2-week2/)

**Week 3:** [Performance Optimization: ElastiCache, Advanced Networking, CloudFront, SSM, and CloudFormation](1.3-week3/)

**Week 4:** [Cost Efficiency with Lambda, Docker/ECR, Advanced Monitoring, and Log Project Design](1.4-week4/)

**Week 5:** [Serverless Development (Lambda, DynamoDB, API Gateway, SAM, CloudFront) and Log Ingestion Setup](1.5-week5/)

**Week 6:** [Advanced Security (Identity Center, Permission Boundary, Security Hub, GuardDuty, WAF) and Log Encryption](1.6-week6/)

**Week 7:** [Advanced Serverless Refactoring: SQS, Athena, Hive Partitioning, and Event-Driven Integration](1.7-week7/)

**Week 8:** [Authentication with Cognito, App Runner Deployment, Full-System Testing, and Project Finalization](1.8-week8/)
