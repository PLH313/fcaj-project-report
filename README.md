# FCAJ Cloud Internship Report

Chào mừng bạn đến với kho lưu trữ báo cáo thực tập của mình trong chương trình **First Cloud AI Journey (FCAJ) Workforce Bootcamp** tại **Amazon Web Services (AWS) Vietnam**.

## Giới thiệu
Đây là nơi ghi lại hành trình học tập và phát triển các kỹ năng về Cloud Computing và DevOps của mình. Báo cáo này bao gồm các công việc thực tế, dự án kỹ thuật và những kiến thức đã tích lũy được trong suốt 2 tháng thực tập.

- **Vị trí:** FCAJ Cloud Intern
- **Chương trình:** First Cloud AI Journey - Workforce Bootcamp
- **Công ty:** Công ty TNHH Amazon Web Services Viet Nam
- **Địa điểm:** Tầng 36, Tòa nhà Bitexco Financial Tower, số 2 Hải Triều, P. Sài Gòn, TP. Hồ Chí Minh.
- **Mentor hướng dẫn:** Nguyễn Gia Hưng
